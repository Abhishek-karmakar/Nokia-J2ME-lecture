This directory contains the example solution and templates for exercises 03-10. 

* * *

TEMPLATES

templates/resources         - all the sounds and images for the game
templates/serversidescripts - server side scripts for the highscore

SOURCE CODE

src/                        - source codes (.java) for the game

* * *

Notice! You must modify files

src/thegame/HighScoreScreen.java
src/thegame/MainScreen.java

to connect to your web server. The default implementation connects to localhost.
